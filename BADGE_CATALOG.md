## Customisation

All badges are generated by `badge_generator.py`. To create a new badge:

```python
svg = make_badge(
    txt="Custom Label",
    accent="#your-color",
    w=220, h=34,
    tag="Tag",
    tag_color="#tag-color",
    dur=5   # glitch cycle in seconds
)
with open("custom-badge.svg", "w") as f:
    f.write(svg)
```

Glitch timing, width, height, accent colour, and tag label are all
independent parameters. The VRIL LABS brand base (dark void background,
CRT scanlines, corner brackets, chromatic aberration) is preserved in
every generated badge.
